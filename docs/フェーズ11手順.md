# フェーズ11:更新手順

アプリファイルのみの更新です(Apps Scriptの変更はありません)。
GitHubで **index.html / styles.css / app.js / sw.js** の4ファイルを上書きし、各端末で開き直してください。

## 今回の変更点まとめ

**今日タブ**
- タイトルをタップして出るチップは、**もう一度同じタイトルをタップすると閉じる**ようになりました(計画タブのチップも同様。自動で消えるまでの時間も4秒に延長)
- チップに**備考(📝)**が表示されます。備考は課題タブのタスク編集で入力でき、備考のあるタスクは一覧にも📝マークが付きます
- タイムラインで**同じ開始時刻のタスクが並ぶ場合、時刻は先頭の1件だけに表示**され、すっきりしました

**計画タブ**
- **カレンダーに予定(●)が1つもないタスクは、名前がオレンジ色**で表示されます(左端にオレンジのバーも付きます)。対象は単発・不定期タスクで、子を持つ行と完了済みは除きます。凡例は ⓘ に追記済みです
- 「今日へ」を押すと、表示範囲だけでなく**日付の選択も今日に**変わります

**課題タブ**
- タスクを保存すると、**保存したタスクの行まで自動スクロール**し、一瞬ハイライトされます。課題に追加した場合はその課題が開いた状態になるので、続けて同じ課題にタスクを足すのが楽になりました
- タスク編集に**備考のテキストエリア**を追加しました(500文字まで)。内容は今日タブ・計画タブのチップで確認できます
