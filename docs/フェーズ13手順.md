# フェーズ13:更新手順

アプリファイルのみの更新です(Apps Script変更なし)。
GitHubで **index.html / styles.css / app.js / sw.js** の4ファイルを上書きし、各端末で開き直してください。

## 今回の変更点まとめ

**今日タブ**
- 超過アラート(音・通知)は**1回の開始につき1回だけ**になりました。中断して再開すると、また1回だけ通知されます(従来の1分ごとの再通知は廃止)
- 「⏳ 同期しています…」バナーが同期完了後も残り続ける不具合を修正しました

**計画タブ**
- チェックボックスを「完了を表示」→「**アーカイブを表示**」に変更。完了していてもアーカイブされていなければ常に表示されます
- **縦スクロールしても日付ヘッダー行が画面上部に残る**ようになりました(未同期バーやミニタイマーが出ている場合はその下に貼り付きます)

**課題タブ**
- 保存後のスクロールを強化:折りたたまれた親の中に保存した場合は自動で展開し、検索・絞り込みで隠れる場合は解除してから、保存した行までスクロール+ハイライトします(未分類への追加、+子での子タスク追加を含む)
- **削除時**も、元あった場所(親→隣のタスク→課題カードの順のアンカー)までスクロールします
- スクロール中にアーカイブボタンがちらつく問題を修正:ボタンはスワイプ中と開いている間だけ表示される方式にしました
- カードのスライドを「枠内で変形」から「**そのまま画面外へはみ出す**」動きに変更しました
- 開いたカードを**右スワイプで戻すときもアニメーション**が付くようになりました(基点を保持する方式に修正)
- **課題カードもタスクと同じ左スワイプでアーカイブ**できます(キャンセル付き通知も同様)。アーカイブ済み課題は「アーカイブ」絞り込みで表示され、📦タグとヘッダーの「解除」ボタンで戻せます。課題をアーカイブすると、その課題のタスクは計画タブから消え、周期タスクの自動予定も止まります
