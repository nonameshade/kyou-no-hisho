# きょうの秘書 — 公開手順

このフォルダの中身をそのままGitHub Pagesに置くだけで、iPhoneから使えるアプリになります。ビルド作業は不要です。所要時間はおよそ15分です。

## 1. GitHubアカウントを作る(持っていない場合)

https://github.com/ にアクセスし、「Sign up」からアカウントを作成します。無料プランで問題ありません。

## 2. リポジトリ(アプリの置き場所)を作る

1. ログイン後、右上の「+」→「New repository」を選ぶ
2. Repository name に `kyou-no-hisho` と入力(好きな名前でも可)
3. 「Public」を選択したまま「Create repository」を押す

## 3. ファイルをアップロードする

1. 作成直後の画面にある「uploading an existing file」というリンクを押す
2. このフォルダの中の全ファイル(index.html、app.js、styles.css、sw.js、manifest.webmanifest、icon-192.png、icon-512.png、icon-180.png)をドラッグ&ドロップする
   - README.md は入れても入れなくても構いません
3. 下の「Commit changes」ボタンを押す

## 4. GitHub Pagesを有効にする

1. リポジトリ画面上部の「Settings」タブ →左メニューの「Pages」を開く
2. 「Branch」のプルダウンで `main` を選び、フォルダは `/ (root)` のまま「Save」を押す
3. 1〜2分待つと、ページ上部に公開URLが表示されます
   - 例: `https://あなたのユーザー名.github.io/kyou-no-hisho/`

## 5. iPhoneのホーム画面に追加する

1. iPhoneの **Safari** で公開URLを開く(Chromeではなく必ずSafariで)
2. 画面下の共有ボタン(四角に上矢印)→「ホーム画面に追加」を選ぶ
3. ホーム画面に「秘書」アイコンが追加され、アプリのように全画面で起動します
4. 一度オンラインで開けば、以降は圏外でも起動・閲覧・タイマー計測ができます

## iPhoneで使う上での注意

- **超過アラートの音**は、iPhoneの消音スイッチがオンだと鳴りません。作業中は消音を解除するか、画面の色変化(全体が赤くなります)を頼りにしてください
- **バイブレーション**はiPhoneのWebアプリでは動作しません(Androidでは動作します)
- **バックグラウンドでの通知**はiPhoneのWebアプリでは制約があります。作業タイマー使用中はアプリを開いたままにしてください。タイマー作動中は画面が自動で消えないようにする仕組み(Wake Lock)を入れています
- データは端末内に保存されます。次のフェーズでGoogleスプレッドシートへの同期を追加します

## アプリを更新するとき

新しいファイルを受け取ったら、リポジトリの「Add file」→「Upload files」から同名ファイルを上書きアップロードするだけで反映されます。iPhoneでは、アプリを一度閉じて2回開き直すと新しいバージョンに切り替わります。
